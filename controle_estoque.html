Controle de Estoque

Este projeto foi desenvolvido em grupo como parte de um projeto semestral da faculdade, com o objetivo de ajudar uma hamburgueria local a organizar melhor o controle de seus produtos.

A ideia surgiu da necessidade de ter um sistema simples e prático para acompanhar os itens disponíveis no estoque, evitando falta de produtos e facilitando a gestão do estabelecimento. Para isso, desenvolvemos uma aplicação web integrada a um banco de dados, permitindo gerenciar as informações de forma organizada.

Close

Funcionalidade

  • Cadastro, edição e exclusão de produtos no sistema.
  • Registro de movimentações de entrada e saída de estoque, organizadas por categorias.
  • Pesquisa e filtros de produtos por nome, categoria e valor.
  • Cálculos automáticos e atualização de valores no sistema.
  • Visão geral com total de produtos e movimentações do mês.

Tecnologias Usadas

  • Python
  • Flask
  • HTML
  • CSS
  • JavaScript
  • MySQL
