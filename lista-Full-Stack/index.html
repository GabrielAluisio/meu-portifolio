Lista de Tarefas - Full Stack

Lista de Tarefas - Full Stack é um projeto pessoal full stack em que utilizei a conexão entre front-end e back-end com integração a banco de dados em uma única aplicação.

No front-end, utilizei HTML, CSS e JavaScript para criar a interface do usuário. Já no back-end, usei Python com Flask para processar as requisições e integrar com um banco de dados MySQL, onde as tarefas podem ser salvas, editadas e excluídas.

Close

Funcionalidade

  • Criar, listar, atualizar e excluir tarefas de forma prática (CRUD completo).
  • API RESTful desenvolvida em Flask para comunicação entre front-end e back-end.
  • Banco de dados MySQL integrado para armazenamento das tarefas.
  • Interface em HTML, CSS e JavaScript conectada diretamente à API.

Tecnologias Usadas

  • Python
  • Flask
  • HTML
  • CSS
  • JavaScript
  • MySQL
